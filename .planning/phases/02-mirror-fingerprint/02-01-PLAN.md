<step number="2" name="define-mirror-types">
<action>
1. Create `crates/wtcd-mirror/src/types.rs` with the following types (all `#[derive(Debug, Clone, Serialize, Deserialize)]`):

```rust
/// Mirror header — maps to mirror-header.schema.json YAML front matter
pub struct MirrorHeader {
    pub anrsm_version: u32,                    // always 1
    pub artifact_type: String,                 // always "file_mirror"
    pub artifact_id: String,                   // "file_mirror:<source_path>"
    pub source_path: String,
    pub source_language: String,
    pub module_id: String,
    pub source_commit: String,
    pub source_fingerprint: String,            // "sha256:<hex>"
    pub semantic_fingerprint: String,          // "sha256:<hex>"
    pub freshness_state: FreshnessState,
    pub confidence_band: MirrorConfidenceBand,
    pub generator_name: String,                // "anrsm"
    pub generator_version: String,             // "0.1.0"
    pub generated_at: String,                  // RFC 3339
    // Optional fields
    pub exports: Option<Vec<String>>,
    pub dependencies: Option<Vec<String>>,
    pub risk_tags: Option<Vec<String>>,
    pub contracts: Option<Vec<String>>,
    pub adjacent_artifacts: Option<Vec<String>>,
    pub manual_appendix_present: Option<bool>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "lowercase")]
pub enum FreshnessState { Fresh, Stale, Invalid, Unknown }

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "lowercase")]
pub enum MirrorConfidenceBand { High, Medium, Low }

/// Complete mirror file content (header + body sections)
pub struct MirrorFile {
    pub header: MirrorHeader,
    pub body: MirrorBody,
}

/// 8 standard mirror sections
pub struct MirrorBody {
    pub responsibilities: String,
    pub external_contracts: String,
    pub preconditions: String,
    pub state_control_flow: String,
    pub side_effects: String,
    pub invariants_risks: String,
    pub change_impact: String,
    pub expand_conditions: String,
}

/// Result of fingerprint computation
pub struct FingerprintResult {
    pub source: String,      // "sha256:<hex>"
    pub semantic: String,    // "sha256:<hex>"
    pub fp_version: String,  // "1"
}
```

2. Add `use` re-exports in `lib.rs`:
   - `pub use types::*;`
3. Verify: `cargo build -p wtcd-mirror` compiles all types
4. Verify: `cargo test -p wtcd-mirror` passes
</action>

       pub output_dir: String,
       #[serde(default = "default_fp_version")]
       pub fp_version: String,
       #[serde(default)]
       pub include_manual_appendix: bool,
