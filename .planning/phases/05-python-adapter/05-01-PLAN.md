<step number="3" name="implement-extract-exports">
<action>
1. Implement `extract_exports(root: &Node, source: &str)` → Vec<Export>
2. Walk the CST for Python export node types:
   - `function_definition` → Export with kind "function"
   - `class_definition` → Export with kind "class"
3. For function_definition:
   - Extract name from `identifier` child
   - Extract parameters from `parameters` child → param names and type annotations
   - Extract return type from `type` child (if present)
   - Extract decorators from parent `decorated_definition` (if present) → decorators field
4. For class_definition:
   - Extract name from `identifier` child
   - Extract base classes from `argument_list` child → bases field
   - Detect `@dataclass` decorator and Pydantic `BaseModel` base
   - Extract decorators from parent `decorated_definition` (if present)
5. Handle __init__.py detection:
   - If file_path ends with "__init__.py", mark in metadata
6. Handle __all__ declarations:
   - Walk for assignment nodes where left side is "__all__"
   - Parse the list of strings on right side
   - Store as __all__ metadata
</action>
